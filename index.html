The Pre-Release Update 1.9.9

skyware-launcher-reloaded

Welcoming the Key + Account ID System

  • To accommodate the plan of Skyware Launcher (Reloaded) entering closed early access soon, a Key and Account ID system has been introduced
  • To gain access to the launcher, you must enter a 32 character Key during Setup (if you already had the launcher prior to v1.9.9, you will be asked to enter a Key on startup)
  • Every Key is linked to your Account ID, which is linked to your specific installation of Skyware Launcher Reloaded (this will be changed to be linked to your Microsoft account instead in Release 2.0)
  • The Account ID makes it easier to connect your launcher to yourself, making researching issues easier - the Account ID does NOT log any of your information whatsoever

Add your own Mods!

  • Introducing the Mods Tab - you can now install and manage your own mods onto Skyware Edition through the Mods Tab, just search the mod you want to add and click Add

In-Game Pack Manager

  • You can now install Resource Packs, Datapacks and Shader Packs right inside Skyware Edition - just enter the Resource Packs/Shader Packs/Data Packs menu in Options and click the plus button on the top right

More Updates

  • Updated all outdated mods to the latest compatible version
